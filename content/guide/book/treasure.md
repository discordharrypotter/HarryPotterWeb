## 스페셜

### 소중한 지팡이 보관함
오색의 광채를 내뿜고 있는 신비로운 보관함입니다. 아마 아주 안전한 보관함이 될 것 같습니다.
- **능력** : 지팡이 보관 가능 (최대 1개)
- **초월 능력** : 22강 미만의 지팡이를 확정 강화 (해당 초월 효과는 비활성화되지 않음)

### 불의 잔
강력한 계약이 성사됩니다. 불의 잔이 타오르는 순간 당신의 이름은 빛날 것입니다.
- **능력** : 프로필 카드를 비공개로 설정
- **초월 능력** : 재료 뽑기에서 20% 확률로 legend 등급 재료가, 80% 확률로 꽝이 등장

### 골든 스니치
재빠르게 날아다니는 이 황금빛 공에 때론 비밀이 숨겨져 있기도 합니다.
- **능력** : 로브에 지팡이 강화 관련 아이템 사용 가능
- **초월 능력** : 재료를 포함한 로브 제작 비용이 50% 감소 (해당 초월 효과는 비활성화되지 않음)

## 전설

### 슬리데린의 로켓
살라자르 슬리데린이 만든 보물로, 직계에게만 전해져 내려오고 있습니다. 이 로켓 안은 절대적으로 안전하다고 합니다.
- **능력** : 방어 마법을 사용했을 때 방어력이 ( 공격력 + LV × 5 ) ÷ 200배 증가
- **초월 능력** : 아즈카반에서 얻는 보상량이 2배 증가

### 후플푸프의 잔
후플푸프의 보물이자 모든 마법이 들어있다는 잔입니다.
- **능력** : 마법 습득 확률이 ( LV × 5 )에 비례하여 증가
- **초월 능력** : 습득 쿨타임이 4초 감소

### 래번클로의 보관
래번클로가 만든 관으로, 지혜가 상승한다고 합니다. 상대의 마법을 파훼할 가능성이 높아집니다.
- **능력** : 상대의 공격력이 ( LV × 5 )% 감소
- **초월 능력** : 전투 시 상대는 공격력이 25% 감소한 상태로 시작

### 그리핀도르의 칼
그리핀도르의 검으로 용기 있는 자만이 뽑을 수 있습니다. 자신에게 이점이 되는 능력은 흡수한다고 합니다.
- **능력** : 전투 시 공격력이 ( LV × 5 )% 증가
- **초월 능력** : 전투 시 공격력이 25% 증가한 상태로 시작

### 딱총나무 지팡이
죽음이 줬다는 최강의 지팡이입니다.
- **능력** : 상대의 남은 체력이 ( LV × 5 )% 이하라면 600 만큼의 피해를 입힘
- **초월 능력** : 해당 보물로 인한 피해량이 ( 현재 공격력 ÷ 4 ) 만큼 증가

### 부활의 돌
죽음에게서 가져왔다는 돌입니다. 죽은 사람을 다시 만날 수 있게 됩니다. 그리운 사람을 다시 만난다면 용기가 나겠죠.
- **능력** : 체력이 0 이하가 되었을 경우, 최대 체력의 ( LV × 5 + 5 )% 만큼의 체력을 가지고 부활
- **초월 능력** : 자신의 부활 체력량이 100%가 됨. 부활 시 해제 가능한 디버프를 모두 해제함

### 투명 망토
죽음에게서도 피할 수 있다는 전설의 망토입니다. 이 망토가 있다면 어떤 적에게서도 피할 수 있습니다.
- **능력** : 회복 마법을 사용했을 때 회복력이 ( 공격력 + LV × 5 ) ÷ 200배 증가
- **초월 능력** : 최대 체력 상태에서 체력을 회복했을 때 최대 체력이 1000 상승

## 레어

### 마법의 모자
호그와트에 있는 모자로 쓴 사람의 자질을 판단하고, 스스로 생각할 수 있는 능력을 가지고 있습니다. 자신의 자질을 안다면 더 강해지기 쉬워지겠죠.
- **능력** : 경험치 ( LV × 10 )% 추가 획득
- **초월 능력** : 경험치 200% 추가 획득

### 마법사의 돌
사람의 생명을 무한히 연장시켜주고, 어떤 물질이든 금으로 만들 수 있다는 돌입니다.
- **능력** : LV% 만큼 습득 쿨타임이 감소
- **초월 능력** : 습득 성공 시 획득 경험치량이 2배 증가

### 딜루미네이터
빛만을 담아두고, 다시 내보내는 정말 신비한 물건입니다. 또한 소유자의 이름을 부를 경우, 부른 위치를 알 수 있게 됩니다.
- **능력** : 자신의 ( 공격력 ÷ 10 + LV ) 만큼 상대의 체력 감소
- **초월 능력** : 전투 시 상대는 자신의 최종 공격력만큼 체력이 감소한 상태로 시작

### 소망의 거울
사람의 소망이 비춰지는 거울입니다. 상대의 욕망을 알 수 있습니다.
- **능력** : 상대의 시작 체력이 LV% 만큼 감소
- **초월 능력** : 전투 시 상대는 체력이 30% 감소한 상태로 시작

### 마법의 눈
한쪽 눈에 낄 수 있는 의안입니다. 360도 전부 볼 수 있으며, 모든 사물을 꿰뚫어 볼 수 있습니다.
- **능력** : LV% 만큼 연구 가속
- **초월 능력** : 연구 완료 시 경험치 획득

### 타임 터너
시간을 되돌아가게 할 수 있습니다.
- **능력** : 5의 배수 턴에 ( 공격력 + LV × 50 ) ÷ 2 만큼 체력을 회복
- **초월 능력** : 5의 배수 턴에 ( 공격력 ÷ 2 ) 만큼 추가로 체력을 회복

### 입학 승인 깃펜과 입학 허가 책
당신이 마법 능력을 발휘했을 때 해당 책에 이름이 적힙니다. 여기엔 강력한 마법 주문이 걸려있어, 당신의 가치를 증명해줍니다.
- **능력** : 매크로 방지 시스템을 해제했을 때 크넛을 획득
- **초월 능력** : 추후 업데이트에서 추가될 예정입니다.

## 일반

### 포트키
위치를 저장한 후, 해당 물건을 잡으면 원하는 위치로 데려다줍니다. 위험에서 빠르게 벗어나세요.
- **능력** : 첫 턴에 방어력이 ( LV × 5 + 50 ) 만큼 상승
- **초월 능력** : 상대의 시작 마나량이 1 감소

### 스니코스코프
위험을 감지합니다. 주변에서 위험이 감지될 경우 시끄러운 소리를 내며 붉은빛을 냅니다.
- **능력** : 자신의 시작 체력이 ( 공격력 ÷ 10 + LV × 2 ) 만큼 증가
- **초월 능력** : 전투 시 체력이 30% 증가한 상태로 시작

### 비밀 탐지기
상대가 거짓말을 하거나 비밀이 있으면 탐지기가 작동합니다.
- **능력** : 보스 몬스터와의 전투에서 승리 시 크넛을 LV% 만큼 추가로 획득
- **초월 능력** : 보스 몬스터와의 전투에서 승리 시 크넛을 10% 만큼 추가로 획득

### 펜시브
자신의 기억을 되돌아보거나, 남들에게 보여줄 수 있습니다. 효율적으로 기억하는 하나의 수단이기도 하죠.
- **능력** : 마법 습득에 실패하면 경험치를 ( LV + 5 ) 만큼 획득
- **초월 능력** : 마법 습득에 실패하면 경험치를 405 만큼 추가 획득

### 리멤브럴
자신이 잊은 것이 있다면 붉은색으로 빛나며 알려주는 도구입니다. 기억력을 향상하는 데 도움이 되겠네요.
- **능력** : 출석체크 보상 중 크넛과 경험치가 LV에 비례하여 증가
- **초월 능력** : 출석체크 보상 중 경험치 보상 최대량이 1.5배가 됨
